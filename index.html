Disk Analyzer

The main objective of this project was to create a visually appealing terminal user interface (TUI) that displays the important system information, similar to a basic version of a task manager to observe system performance, while maintaining simplicity.

The following tool is a system tool, which monitors different aspects (i.e. CPU usage, disk space, network speed, etc). This project was done in Python using the library 'Rich', which is a TUI that displayed the specifications seen below.

The sections 'CPU Usage' and 'Network Speed' are updated in real-time. Moreover to get systems details, the 'psutil' library was used to display information for the sections 'Disk Space', 'CPU Info', 'System specifications' and'Other Specifications'.
